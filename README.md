📚 C++ Fundamentals: Data Structures & Algorithms~ 基礎から積み上げるコンピュータサイエンスの構造化 ~このリポジトリは、現代のソフトウェア開発の基盤となるデータ構造とアルゴリズムを、C++を用いてスクラッチから実装したコレクションです。STL（標準ライブラリ）に頼りすぎず、ポインタ操作やメモリ管理を直接記述することで、ロジックの透明性と効率性を追求しています。🌟 プロジェクトの概要「車輪の再発明」は、エンジニアの基礎体力を養うために不可欠です。本プロジェクトは、データの流れ（LIFO/FIFO）や探索の最適化、再帰による複雑な問題の解決など、アルゴリズムの本質的な構造を可視化しています。✨ 搭載コンポーネント1. 高速ソート & 探索 (Sort & Search):Quick Sort: ピボット選択と分割統治による $O(n \log n)$ のソート。Binary Search: ソート済みデータに対する対数時間 $O(\log n)$ の高速検索。2. スタック & キュー (LIFO & FIFO):MyStack (LIFO): 動的なリンクリストによるスタック。MyQueue (FIFO): front / rear ポインタを用いた効率的なキュー。最後尾への追加と先頭からの取り出しを $O(1)$ で実行。3. 木構造 (Binary Search Tree):二分探索木の構築。再帰による中間順巡回（In-order）を実装し、構造化されたデータ走査を実証。4. 再帰的思考 (Tower of Hanoi):複雑なステップを最小単位の移動に分解する、数学的帰納法の実装。5. 文字列パターンマッチング (Regex):std::regex を用いたメールアドレスのバリデーション。実務的なテキスト処理への応用。🛠 技術スタックLanguage: C++ 17+Core Skills:動的メモリ管理 (new / delete)リンクリストによるカスタムコンテナポインタによる境界条件制御再帰的プログラミング🚀 実行方法📝 実装のハイライト: MyQueueキューの実装では、末尾ポインタ（rear）を保持することで、要素の追加（Enqueue）を定数時間で実現しています。また、dequeue 時にキューが空になる瞬間のポインタ処理（rear = nullptr）など、堅牢なエッジケース処理を行っています。📜 ライセンス「優れたコードは、優れたデータ構造の選択から始まる。」
