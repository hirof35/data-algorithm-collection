📚 C++ Algorithm & Data Structure Core~ コンピュータサイエンスの基礎構造を再構築する ~C++ Algorithm & Data Structure Core は、効率的なソフトウェア開発の根幹をなす主要なアルゴリズムとデータ構造を、C++でスクラッチから実装したコード・ライブラリです。🌟 プロジェクトの概要「ライブラリの中身がどう動いているか」を理解することは、最適で堅牢なシステムを設計する上で不可欠です。本プロジェクトでは、STLの抽象化に隠されたロジックを可視化し、メモリ管理（ポインタ操作）から再帰的思考まで、プログラミングの本質的なスキルを網羅しています。✨ 搭載されている主要コンポーネント1. 高速ソート & 探索 (Sort & Search):Quick Sort: 分割統治法を用いた $O(n \log n)$ の効率的なソートアルゴリズム。Binary Search: ソート済み配列に対する $O(\log n)$ の高速検索。2. 線形データ構造 (Dynamic Stack):StackNode 構造体を用いたリンクリストベースのスタック。動的なメモリ割り当てと、LIFO（後入れ先出し）の原理を実装。3. 木構造 (Binary Search Tree):二分探索木（BST）の構築と、中間順巡回（In-order Traversal）によるデータのソート済み出力。4. 再帰的アルゴリズム (Tower of Hanoi):数学的帰納法に基づいたパズル解法。再帰呼び出しによる「問題の最小化」を実証。5. テキスト処理 (Regular Expressions):std::regex を用いたメールアドレスのバリデーション。正規表現による高度なパターンマッチングの実装。🛠 技術スタックLanguage: C++ 17 (or later)Concepts:分割統治法 (Divide and Conquer)メモリ管理 (new/delete によるポインタ制御)再帰 (Recursion)正規表現 (Regex)🚀 実行方法C++17 以上に対応したコンパイラ を用意します。以下のコマンドでビルドおよび実行を行います。📝 実装の詳細メモリの安全性: 自作スタックでは pop 時にしっかりと delete を行い、メモリリークを防ぐ実装になっています。論理の可視化: ハノイの塔の実行ログにより、複雑な再帰処理の推移をステップバイステップで確認できます。📜 ライセンス「アルゴリズムを知ることは、論理の迷路を解く地図を持つことに等しい。」
